# Portfolio fotográfico para GitHub Pages

## Cómo usarlo

1. Cambia el nombre "Agustín Andorno" por tu nombre si quieres.
2. Cambia `tuemail@gmail.com` y `@tuusuario`.
3. Sustituye las imágenes dentro de:
   - images/project-1/
   - images/project-2/
   - images/project-3/
4. Mantén los nombres:
   - foto1.jpg
   - foto2.jpg
   - foto3.jpg
   - foto4.jpg
5. Sube todo a un repositorio llamado `tuusuario.github.io`.
6. Activa GitHub Pages desde Settings → Pages.

## Consejos

- Usa fotos JPG de máximo 2000px en el lado largo.
- Empieza con 3 proyectos y 4 fotos por proyecto.
- Cuando funcione, puedes duplicar páginas de proyecto y añadir más fotos.
